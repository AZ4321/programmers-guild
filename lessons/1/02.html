This session will introduce students to open-source and the open-source curriculum for the Guild!

Learning Objectives:
  • Explain what open-source is
  • Explain how to fork and clone a repository, and make a pull-request
  • Suggest future guild activities
Agenda:
0000 - 0010 Arrival and Registration
0010 - 0110 Exploring Open-Source Culture
0110 - 0130 Introduction to the Open-Source Curriculum
0130 - 0150 Programming Survey
0150 - 0210 Coffee Break
0210 - 0245 General Programming Support and Discussion
0245 - 0250 Debrief
0250 - 0300 Departure

Further Guidance:
  • Read What Is Free Software?
  • Read History of the Open Source Initiative
  • Analyse the Wikipedia Page on Open-source Software
  • Read The Cathedral and the Bazaar
  • Read 50 Ways to be a FOSSer

  • Fork the Programmer's Guild Repository
  • Clone the repository on to your local machine using TortoiseGIT
  • If you're unsure how to do this, review our Brief Video on Version Control
  • Staff and Guild Champions are the 'Maintainers' of the Repository
  • They will be able to accept and/or request revisions and/or reject pull requests
  • Research a topic that you're interested in, and plan a session for the Programmer's Guild
  • If you're not familiar with HTML, then review the materials on W3Schools

  • Everyone should make a proposal using the pull-request mechanism
  • Guild champions will review these and discuss with staff
Further Learning:
  • Complete a FOSS Field Trip
  • Review an Open-Source Educational Game and Repository
  • Review an Active HFOSS Project
  • Research HFOSS
Resources:
  • Free Software Foundation - http://fsf.org/
  • Open Source Initiative - http://opensource.org/
